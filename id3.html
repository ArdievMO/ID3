Расчет информативности узлов

            

                
Математические формулы для определения итоговой оценки атрибута

                

                    
1. Энтропия множества примеров S (целевой атрибут):

                    
H(S) = −∑i pi · log₂(pi)

                    
где pi — доля примеров класса i в S. Энтропия равна нулю, если все примеры одного класса.

                

                

                    
2. Взвешенная энтропия по атрибуту A:

                    
H(S|A) = ∑v∈Values(A) (|Sv| / |S|) · H(Sv)

                    
где Sv — подмножество примеров с значением атрибута A = v.

                

                

                    
3. Прирост информации (Information Gain):

                    
Gain(S, A) = H(S) − H(S|A)

                    
Чем больше прирост информации, тем полезнее атрибут для разбиения.

                

                

                    
Итоговая оценка и выбор атрибута:

                    
Abest = argmaxA Gain(S, A) = argminA H(S|A)

                    
На каждом шаге ID3 выбирает атрибут с максимальным приростом информации (или минимальной взвешенной энтропией).
